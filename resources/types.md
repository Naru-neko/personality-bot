# MBTI TYPES 

## 外向型 (E) / 内向型 (I)

* **外向型 (E)**：外界からエネルギーを得る
* **内向型 (I)**：内面からエネルギーを得る

## 感覚型 (S) / 直観型 (N)

* **感覚型 (S)**：現実的な事実や経験を重視
* **直感型 (N)**：可能性やアイデアを重視

## 思考型 (T) / 感情型 (F)

* **思考型 (T)**：論理や客観性を重視
* **感情型 (F)**：価値観や主観性を重視

## 判断型 (J) / 知覚型 (P)

* **判断型 (J)**：計画性や整理整頓を重視
* **知覚型 (P)**：柔軟性や臨機応変さを重視
